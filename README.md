# Сравнительный анализ устойчивости городских топологий к транспортным коллапсам

## Цель исследования
Анализ степени устойчивости четырех различных типов городский топологий к транспортным коллапасам, сравнение и выявление наиболее устойчивой топологии

## Объекты исследования
1. **Москва** - представитель кольцевой структуры
2. **Нью-Йорк** - представитель сетчатой структуры
3. **Волгоград** - представитель линейной структуры
4. **Лондон** - представитель смешанной / хаотичной структуры

## Метрики оценки
1. Размер самого большого связного кластера
2. Средний размер изолированных кластеров
3. Топологический диаметр
4. Реальный диаметр (в метрах)
5. Глобальная эффективность
6. Коэффициент локальной кластеризации

## Типы воздействий
1. **Ошибка** - выведение из строя случайного узла
2. **Атака** - целенаправленное выведение узла с наибольшей степенью
